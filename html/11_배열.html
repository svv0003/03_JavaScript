배열(Array)

      - 변수를 묶음으로 다루는 것 (자료 구조의 기초)
        (JS 변수의 특징 : 대입되는 값에 따라서 자료형이 변함)

      -> JS 배열 : 자료형 상관 없이 여러 값을 한 번에 다룬다.

      - JS 배열 특징
      1. 자료형 제한 X (하지만 보통은 한 가지 자료형만 저장한다.)
      2. 길이 제한 X (자동으로 배열 길이가 늘었다, 줄었다 한다.)
      
      [JS 배열 선언 방법]
      + [] : 배열 기호
      1. const arr = new Array();       // 0칸짜리 배열(비어있는 배열) 생성
      2. const arr = new Array(3);      // 3칸짜리 배열 생성 (칸은 비어있는 상태)
      3. const arr = [값1, 값2, 값3];    // 3칸짜리 배열 생성 (값으로 채워진 상태)
      4. const arr = [];                // 0칸짜리 배열(비어있는 배열) 생성
