이벤트 e, event, value, target

1. e와 event는 같은 것이다!
  e와 event는 모두 이벤트 객체이다.


function 기능1(e) {
  // 기능1에 해당하는 기능 내용들
}

function 기능2(event) {
  // 기능2에 해당하는 기능 내용들
}
예제1 : e와 event는 동일하다.

function 클릭처리1(e) {
  console.log("e 사용 : ", e.type);
}

function 클릭처리2(event) {
  console.log("event 사용 : ", event.type);
}
버튼을 클릭하세요.
예제2 : target 사용법

function 타겟확인(e) {
  console.log("클릭된 요소 : ", e.target.tagName);
  console.log("요소 내용 : ", e.target.textContent);
}

예제3 : value 사용법

function 값확인(e) {
  console.log("입력 값 : ", e.target.value);
  console.log("입력 타입 : ", e.target.type);
}
