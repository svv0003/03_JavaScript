조건문(condition)

      프로그래밍 코드가 해석되는 순서를 제어하는 제어문의 한 종류
      작성된 조건식에 따라서 수행되는 코드를 변경하는 구문

if(만약에...)

      
        if (조건식) {
          조건식이 true인 경우에 실행할 코드;
        }
      
      * 조건식 : 결과가 true, false인 식(연산)
      * 조건문 : 조건식의 결과에 따라 실행을 제어하는 구문(if문, switch문)

입력 받은 수가 양수인지 아닌지 검사(1)


입력 받은 수가 양수인지 아닌지 검사(2)


if(만약에) - else if(아니라면 이런 경우에) - else(모두 아니라면)

      
        if(조건식1) {
          조건식1이 true인 경우에만 실행
        }
        else if(조건식2) {
          조건식1이 false, 조건식2가 true인 경우에만 실행
        }
        else {
          위 모든 조건식이 false인 경우에만 실행
        }
      
      * else if 구문은 원하는 만큼 작성 가능하다.
      * if문, else if문 조건식이 falseㅇ니 경우
        이후 else if문에서 해당 범위는 제외된다.

어린이, 청소년, 성인 구분하기


switch, case, default

      switch문
      - 식 하나의 결과로 많은 경우의 수를 처리할 때 사용하는 조건문
        -> 식의 결과에 맞는 case 구문이 수행됨

      
        switch(식) {
          case 결과1 : 결과1의 수행코드; break;
          case 결과2 : 결과2의 수행코드; break;
          case 결과3 : 결과3의 수행코드; break;
          ...
          default : 모든 case를 만족하지 못할 때 수행하는 기본 코드;
        }
